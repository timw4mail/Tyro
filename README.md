#Tyro

A Cross-platform Code Editor

### Planned Features

* Syntax Highlighting
* Line Numbers
* Custom Syntax themes
* SFTP editing

### Building
* Make sure wxWidgets is installed, version 2.8 or 3

#### Mac and Linux
Use either the Netbeans project, or the makefile. 

On OS X, `make Tyro.app` will create the app bundle.
